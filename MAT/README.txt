Scripts for formatting the input fossile files as required by Joel's scripts for running mat (also works for package 'analogue'),
and for running MAT reconstructions in a batch using package 'analogue' (i.e. a number of fossil files/cores one by one).
Uses the n=1968 database of dinocyst assemblage data dino1968.txt 
and cross-referenced environmental parameters envi1968.txt
Citation for n=1968 database: https://doi.pangaea.de/10.1594/PANGAEA.908494.
