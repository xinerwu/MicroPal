For post-processing age-model using package "rbacon" (https://cran.r-project.org/web/packages/rbacon/index.html). 
See reference mannual here: https://cran.r-project.org/web/packages/rbacon/rbacon.pdf
The script interpolates ages at given sample depths from the output of rbacon (XXXXX_NN_ages.txt).
Need to prepare a file containing sample IDs and depths following sample_depths.xlsx
After running the script, interpolated ages will be write back to sample_depths.xlsx
